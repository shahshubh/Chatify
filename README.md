## Features

* User Authentication.
* Realtime chat.
* Send text messages, stickers, GIFs & Images.
* Permanently delete messages.
* View recent chats.
* View user status online / offline / Lastseen.
* Notifications on new message.
* Video calling.
* View / delete the call logs.
